🤖 Présentation

BorzBot est un bot Discord de modération, tickets et configuration serveur. Il permet de gérer plus facilement un serveur Discord avec des outils propres et rapides.

  • Commandes de modération
  • Système de tickets
  • Logs de modération
  • Anti-link, anti-spam et anti-insulte
  • Configuration par serveur
  • Dashboard en développement

🚀 Installation

Pour installer BorzBot, invite le bot sur ton serveur avec les bonnes permissions.

⚠️ Le rôle de BorzBot doit être placé au-dessus des rôles qu’il doit gérer.

Permissions recommandées

  • Gérer les messages
  • Gérer les rôles
  • Gérer les salons
  • Expulser des membres
  • Bannir des membres
  • Utiliser les commandes d’application

⚙️ Configuration

/setupbot

Configure les éléments principaux du bot.

/ticketsubjects add

Ajoute un sujet de ticket avec un emoji, une description et un salon de logs.

/ticketsubjects list

Affiche les sujets configurés.

/sendticketpanel

Envoie le panel d’ouverture de ticket.

/setlinks

Configure les liens importants.

/sendlinks

Envoie l’embed des liens utiles.

🎫 Tickets

Le système de tickets permet aux membres de contacter le support facilement.

/ticketsubjects add label:Bug emoji:🐞 description:Signaler un bug log:#logs-bug

Quand un membre choisit un sujet, BorzBot crée automatiquement un salon privé avec les bonnes permissions.

❌ Problèmes fréquents

Le bot ne répond pas

Vérifie que le bot est en ligne, que les commandes slash sont déployées et que les permissions sont bonnes.

Les tickets ne se créent pas

Vérifie la catégorie ticket, le rôle staff et les sujets configurés.

Erreur MongoDB

Vérifie ton lien MongoDB dans le fichier .env.
